# MediAssist

Hackathon prototype for an offline-first healthcare guidance assistant.

## Current Build
- Flutter Android-first prototype
- Telugu and English text guidance
- Offline symptom rules from local JSON
- Red-flag emergency escalation
- Offline and online mode toggle (online enrichment currently simulated)

## Run
1. Open terminal in project root.
2. Run:

```bash
cd app
flutter pub get
flutter run
```

## Important Medical Boundary
This prototype provides basic guidance only and is not a diagnosis tool.
